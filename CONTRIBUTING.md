# Contributing to the AI Manga Generator Project

Thank you for taking the time to contribute to our project! We welcome contributions from everyone, especially beginners. Here are some guidelines to get you started:

## Getting Started
1. **Fork the Repository:**  Click on the "Fork" button on the top right corner of this repository page.
2. **Clone Your Fork:**  Use the following command to clone your fork to your local machine:
   ```bash
   git clone https://github.com/YOUR_USERNAME/AI-manga-generator.git
   ```
3. **Set Upstream Repository:** After cloning, set the original repository as the upstream source:
   ```bash
   git remote add upstream https://github.com/mrgeekay727/AI-manga-generator.git
   ```

## Making Changes
1. **Create a New Branch:**  Before making changes, create a new branch:
   ```bash
   git checkout -b feature/name-of-your-feature
   ```
2. **Make Changes:**  Now, you can start making your changes.  Make sure to follow the existing code style.
3. **Commit Your Changes:**  Once you are satisfied with your changes, add and commit them:
   ```bash
   git add .
   git commit -m "Description of changes"
   ```

## Submit a Pull Request
1. Push your changes to your forked repository:
   ```bash
   git push origin feature/name-of-your-feature
   ```
2. Go to the original repository, and click on the "New Pull Request" button.
3. Fill out the form, providing details about your changes, and submit the pull request.

## Code of Conduct
We expect all contributors to adhere to the [Code of Conduct](CODE_OF_CONDUCT.md).

## Questions?
If you have any questions, feel free to ask! We're here to help you out. Happy coding!
